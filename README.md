# 計算機網路 Ch02 — 模擬考 100 題

這是一個計算機網路第二章的模擬考試題目集，包含 100 題選擇題和答案。

## 功能特色

- 📝 100 題模擬考試題目
- ✅ 包含完整答案
- 🎯 可點擊題目顯示/隱藏答案
- 📱 響應式設計，支援各種裝置
- 🖨️ 支援列印功能

## 使用方式

1. 直接在瀏覽器中開啟 `index.html`
2. 點擊「顯示答案」按鈕可展開所有答案
3. 點擊單個題目可切換該題答案的顯示/隱藏
4. 使用「隱藏所有」按鈕可收起所有答案

## 部署到 GitHub Pages

### 自動部署（推薦）

本專案已配置 GitHub Actions，當您推送代碼到 `main` 分支時會自動部署。

### 手動啟用 GitHub Pages

1. 前往您的 GitHub 倉庫：https://github.com/brycee1628/take-an-exam
2. 點擊 **Settings**（設定）
3. 在左側選單中找到 **Pages**
4. 在 **Source** 部分：
   - 選擇 **GitHub Actions**（如果使用自動部署）
   - 或選擇 **Deploy from a branch**，然後選擇 `main` 分支和 `/ (root)` 目錄
5. 點擊 **Save**（儲存）

部署完成後，您的網站將在以下網址可用：
`https://brycee1628.github.io/take-an-exam/`

## 題目涵蓋範圍

- 儲存單位與傳輸速度
- 有線傳輸媒介（雙絞線、同軸電纜、光纖）
- IEEE 802 標準
- CSMA/CD 與 Ethernet Frame
- Ethernet 規格（10Base-T、100Base-TX、1000Base-T 等）
- Gigabit 技術
- 5-4-3 規則
- 調變技術（PCM、ASK、FSK、PSK、QAM）
